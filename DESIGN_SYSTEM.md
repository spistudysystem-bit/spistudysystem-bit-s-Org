# Design System & Color Palette

Here are the specific colors used in the application. You can copy these hex codes for your own use.

## Brand Colors

| Color Name        | Hex Code  | Tailwind Class          | Usage                                      |
|-------------------|-----------|-------------------------|--------------------------------------------|
| **Primary Blue**  | `#003057` | `bg-echomasters-blue`   | Main headers, primary buttons, borders     |
| **Dark Blue**     | `#00264d` | `bg-echomasters-dark`   | Mobile menus, button hover states          |
| **Darker Blue**   | `#001a33` | `bg-echomasters-darker` | Button borders (3D effect)                 |
| **Light Blue**    | `#004070` | `bg-echomasters-light`  | Input backgrounds, secondary states        |

## Accent Colors

| Color Name        | Hex Code (Approx) | Tailwind Class | Usage                                      |
